  <section class="section" id="warto">
  <h2>Dlaczego warto trenować siłowo?</h2>
  <p>Trening siłowy przynosi wiele korzyści dla ciała i umysłu. Oto najważniejsze z nich:</p>
  <ul>
    <li><strong>Budowa mięśni</strong> – poprawa sylwetki i siły fizycznej.</li>
    <li><strong>Przyspieszenie metabolizmu</strong> – łatwiejsze spalanie kalorii i kontrola wagi.</li>
    <li><strong>Wzmocnienie stawów i kości</strong> – mniejsze ryzyko kontuzji i lepsza stabilność.</li>
    <li><strong>Poprawa postawy</strong> – silniejsze mięśnie głębokie i mniej bólu pleców.</li>
    <li><strong>Korzyści psychiczne</strong> – redukcja stresu, lepszy nastrój i większa pewność siebie.</li>
    <li><strong>Lepsze zdrowie metaboliczne</strong> – niższe ryzyko cukrzycy typu 2 i chorób serca.</li>
    <li><strong>Długowieczność i sprawność</strong> – trening siłowy opóźnia procesy starzenia się organizmu.</li>
  </ul>
</section>

  <section class="section" id="technika">
    <h2>Technika ćwiczeń – podstawa bezpieczeństwa i efektywności</h2>
    <p>Jednym z najważniejszych aspektów treningu siłowego jest prawidłowa technika wykonywania ćwiczeń. Nieprawidłowa forma może prowadzić do kontuzji i znacznie obniżyć efektywność treningu. Dlatego przed rozpoczęciem treningu warto poświęcić czas na naukę prawidłowego ustawienia ciała, zakresu ruchu oraz kontroli nad ciężarem.</p>
    <p>Dobrym sposobem na opanowanie techniki jest korzystanie z porad profesjonalnych trenerów, oglądanie sprawdzonych filmów instruktażowych oraz ćwiczenie przed lustrem lub z partnerem, który może zwrócić uwagę na błędy. Na początku warto wybierać mniejsze obciążenia, by skupić się na jakości ruchu, a dopiero później stopniowo zwiększać ciężary. Pamiętaj też o odpowiednim rozgrzewaniu się przed treningiem i rozciąganiu po ćwiczeniach, co pomaga zapobiegać urazom i poprawia regenerację mięśni.</p>
  </section>

  <section class="section" id="partie">
  <h2>Jakie partie ciała ćwiczyć, by wyglądać estetycznie?</h2>
  <p>Estetyczna sylwetka to nie tylko duże mięśnie, ale przede wszystkim proporcje i równomierny rozwój ciała. Zamiast skupiać się wyłącznie na jednej partii (np. klatce piersiowej czy bicepsach), warto zadbać o wszystkie kluczowe grupy mięśniowe. Oto partie, na które warto zwrócić szczególną uwagę:</p>
  <ul>
    <li><strong>Plecy (szczególnie najszersze grzbietu)</strong> – szerokie plecy tworzą iluzję wąskiej talii i dają kształt litery V.</li>
    <li><strong>Barki (przede wszystkim boczne aktony)</strong> – dobrze rozwinięte barki poszerzają sylwetkę i dodają atletycznego wyglądu.</li>
    <li><strong>Klatka piersiowa</strong> – pełna klatka nadaje objętości górnej części ciała i poprawia postawę.</li>
    <li><strong>Nogi (uda i pośladki)</strong> – proporcjonalna dolna część ciała podkreśla sylwetkę i pokazuje ogólną sprawość fizyczną.</li>
    <li><strong>Mięśnie brzucha</strong> – nie tylko dla estetyki, ale także dla stabilizacji i zdrowej postawy.</li>
    <li><strong>Ramiona (biceps i triceps)</strong> – jako dodatek, podkreślają muskulaturę, zwłaszcza przy dobrze rozwiniętych barkach i plecach.</li>
  </ul>
  <p>Estetyczna sylwetka to efekt konsekwentnej pracy nad całym ciałem, a nie tylko nad „widocznymi” mięśniami. Dbaj o balans, technikę i odpowiednią regenerację, a rezultaty przyjdą szybciej, niż się spodziewasz.</p>
</section>

    <h2>Regeneracja – sekret postępów</h2>
    <p>Regeneracja jest kluczowym elementem każdego programu treningowego. Mięśnie rosną nie podczas samego treningu, ale w czasie odpoczynku. Dlatego warto zadbać o odpowiednią ilość snu (7-9 godzin na dobę), unikać przeciążeń i stosować techniki relaksacyjne, takie jak masaż czy stretching.</p>
    <p>Ważna jest też dieta bogata w białko, witaminy i minerały, które wspomagają odbudowę tkanek. Nie bój się także robić dni wolnych od treningu – pozwoli to uniknąć przetrenowania i kontuzji.</p>

    <script>
    document.addEventListener("DOMContentLoaded", () => {
    const calculateBtn = document.getElementById('calculateBtn');
    const weightInput = document.getElementById('weight');
    const heightInput = document.getElementById('height');
    const resultBox = document.getElementById('resultBox');

    calculateBtn.addEventListener('click', () => {
      const weight = parseFloat(weightInput.value);
      const height = parseFloat(heightInput.value) / 100;

      if (!weight || !height || height <= 0) {
        resultBox.textContent = 'Wprowadź poprawne dane.';
        resultBox.style.backgroundColor = '#f8d7da';
        resultBox.style.color = '#721c24';
        return;
      }

      const bmi = (weight / (height * height)).toFixed(2);
      let category = '';
      let bgColor = '';
      let textColor = '#fff';

      if (bmi < 18.5) {
        category = 'Niedowaga';
        bgColor = '#f1c40f';
        textColor = '#000';
      } else if (bmi < 24.9) {
        category = 'Waga prawidłowa';
        bgColor = '#2ecc71';
      } else if (bmi < 29.9) {
        category = 'Nadwaga';
        bgColor = '#f39c12';
      } else {
        category = 'Otyłość';
        bgColor = '#e74c3c';
      }

      resultBox.textContent = `Twoje BMI: ${bmi} (${category})`;
      resultBox.style.backgroundColor = bgColor;
      resultBox.style.color = textColor;
    });
  });
</script>
